# Contributing to Nexus-Prime

Thank you for your interest in contributing to the Nexus-Prime Enterprise Command Center! 

## Core Focus
Nexus-Prime is designed to be a lightweight, high-performance visualization tool. When contributing, please focus on:
1.  **Canvas Performance**: Ensuring the 3D globe remains smooth even on lower-end hardware.
2.  **UI/UX Consistency**: Maintaining the glassmorphic design system.
3.  **Modular Logic**: Keeping the telemetry modules decoupled and easy to extend.

## Contribution Workflow
1.  Fork the repository.
2.  Create a feature branch.
3.  Submit a Pull Request with a clear description of your changes and why they are necessary.

## Style Guide
*   Maintain the **Vanilla JavaScript** philosophy.
*   Document your Canvas-drawing functions clearly.
